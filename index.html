00The tutorial & setup

The heart of the workshop is a single hands-on notebook in two parts. You'll run it on a free Colab/Kaggle GPU — no local install or cluster needed.

Run-it-yourself notebook

BDBio 2026 — LLMs in Genomics

From loading a tiny DNA language model to fine-tuning a 500M-parameter foundation model on real regulatory-genomics tasks — start to finish on a free GPU.

Part 1 · Using DNA language models

Reading & scoring DNA

  • Load a toy model (Mistral-DNA-v1-17M) and its tokenizer
  • Compute sequence embeddings and count parameters
  • Score a real CFTR cystic-fibrosis mutation by log-likelihood
  • Explore the hosted Evo 2-40B and NVIDIA Protein Design interfaces

Part 2 · Task-specific fine-tuning

LoRA on Nucleotide Transformer

  • Adapt NT-500M-human-ref with PEFT / LoRA
  • Task 1 — promoter prediction (F1 ≈ 0.94 in ~1000 steps)
  • Task 2 — enhancer prediction, scored by MCC on imbalanced data
  • Compare against the published Nucleotide Transformer results

01Foundation models

The large "sequence-to-function" and generative models. Mostly used via API or inference rather than trained from scratch, but they define the frontier.

DeepMind · sequence→function

AlphaGenome

Takes up to 1 Mb of DNA and predicts thousands of regulatory tracks (expression, accessibility, splicing, TF binding) at base resolution, and scores variant effects. Published in Nature (Jan 2026); free non-commercial API, source now open.

Overview · GitHub · Nature

Arc Institute · generative

Evo 2 tutorial

A 40B-parameter DNA/RNA/protein model with a 1 Mb context (StripedHyena 2), trained on 9.3T nucleotides across >128k species. Generalist prediction and design; published in Nature (Mar 2026); fully open via Arc & NVIDIA BioNeMo.

Project · GitHub · Evo Designer

DeepMind · regulatory

Enformer

The influential predecessor: a transformer predicting gene expression and chromatin from ~200 kb of sequence. A clean entry point for "DNA → function" intuition.

HF model · Paper

Calico · regulatory

Borzoi

Predicts RNA-seq coverage directly from sequence — a strong open successor to Enformer and one of AlphaGenome's main external baselines.

GitHub · Paper

02DNA language models you can fine-tune

Small-to-medium models that load on a free GPU and fine-tune in minutes-to-hours with LoRA/PEFT. All are on the Hugging Face Hub.

ModelWhat it isHub / code
Nucleotide Transformer tutorial InstaDeep's BERT-style genomic models (up to 2.5B). Strong baseline across 18 tasks; fine-tuned with PEFT/LoRA. HF · code · leaderboard
DNABERT-2 Efficient multi-species BERT with byte-pair tokenisation. Ships the GUE benchmark suite. A solid teaching default. HF · code · paper
HyenaDNA Convolution-based long-context model (single-nucleotide, up to ~1M tokens). Good contrast to attention. HF · code · paper
Mistral-DNA tutorial Mistral/Mixtral adapted to DNA, sizes 17M–422M. Repo includes ready-to-run fine-tuning notebooks. HF · code
Caduceus State-space (Mamba) bi-directional, reverse-complement-equivariant DNA model. HF · code
GENA-LM Long-context BERT-style human genomic models from AIRI. HF · code

Tip for the live session: pick one model + one binary task and run the whole load → tokenise → LoRA → evaluate loop end-to-end — exactly what Part 2 of the tutorial does.

03Hands-on & fine-tuning

The practical toolkit. If students learn one thing technically, it should be parameter-efficient fine-tuning (LoRA) with the Hugging Face Trainer.

Library

PEFT (LoRA)

Low-rank adaptation: freeze the base model, train tiny adapter matrices. This is what makes fine-tuning a genomic LM feasible on a free GPU.

Docs · GitHub · LoRA paper

Course · free

Hugging Face LLM / NLP Course

The clearest free introduction to tokenisers, the Trainer API, datasets and fine-tuning. The mechanics transfer directly to DNA models.

huggingface.co/learn

Course · audit free

Generative AI with LLMs

DeepLearning.AI & AWS course on how LLMs actually work — transformer internals, prompting, fine-tuning, PEFT/LoRA and RLHF. A solid conceptual backbone for the genomics tutorial.
